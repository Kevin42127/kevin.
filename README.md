# Kevin. - 個人作品集網站

運用 AI 協作快速打造智能應用，專注使用者體驗，將技術轉化為實用的數位產品。

## 🌟 特色功能

- **🌐 雙語支援**：完整的中英文國際化
- **🤖 AI 助理**：智能對話系統，提供即時協助
- **📱 PWA 支援**：可安裝為獨立應用程式
- **🎨 現代化設計**：響應式佈局，深淺主題切換
- **⚡ 高效能**：Next.js 14 + React 18 + TypeScript

## 🛠️ 技術棧

- **前端框架**：Next.js 14.2.32
- **UI 庫**：React 18 + Framer Motion
- **樣式系統**：Tailwind CSS + CSS Variables
- **語言**：TypeScript
- **AI 整合**：Groq API
- **部署**：Vercel

## 📂 專案結構

```
src/
├── app/                 # Next.js App Router
├── components/          # React 組件
├── lib/                # 工具函數與配置
├── hooks/              # 自定義 Hooks
└── types/              # TypeScript 類型定義
```

## 🚀 快速開始

1. **安裝依賴**
   ```bash
   npm install
   ```

2. **設定環境變數**
   ```bash
   cp .env.example .env.local
   ```

3. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

4. **開啟瀏覽器**
   ```
   http://localhost:3000
   ```
